# SnapLog GitHub Pages

This folder contains the GitHub Pages site for SnapLog.

## Setup Instructions

1. **Update GitHub Username**: 
   - Open `index.html` and replace all instances of `YOUR_USERNAME` with your actual GitHub username
   - Update the download links to match your repository name

2. **Add Demo GIF**:
   - Place your demo GIF file as `demo.gif` in this folder
   - The page will automatically display it

3. **Enable GitHub Pages**:
   - Go to your repository Settings â†’ Pages
   - Under "Source", select "Deploy from a branch"
   - Choose the `main` branch (or your default branch)
   - Select the `/docs` folder
   - Click Save

4. **Your site will be available at**:
   - `https://YOUR_USERNAME.github.io/snaplog/`

## Files

- `index.html` - Main landing page
- `logo-universal.png` - SnapLog logo
- `demo.gif` - Demo GIF (add this file)

## Customization

The page uses a clean, modern design with:
- Natural color scheme (blue primary, green accents)
- Responsive layout
- Smooth animations
- Professional typography

You can customize colors by editing the CSS variables in the `<style>` section of `index.html`.
